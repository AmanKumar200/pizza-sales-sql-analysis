# 🍕 Pizza Sales SQL Analysis Project

## 📌 Overview

Businesses like Pizza Hut generate large volumes of transactional data daily, but raw data alone does not provide actionable insights. The key challenge is to transform this unstructured data into meaningful business intelligence that can support decision-making.

In this project, I worked with a pizza sales dataset containing information on orders, pizza types, pricing, and quantities. The dataset lacked direct insights into revenue drivers, customer preferences, and time-based demand patterns.

To solve this, I used SQL to clean, join, and analyze the data, converting it into structured insights that answer critical business questions such as:
- Which pizzas generate the most revenue?
- What are the peak order times?
- Which categories perform best?
- How does revenue grow over time?

This project demonstrates how raw transactional data can be transformed into actionable insights using SQL.

---

## 🎯 Objectives

The primary goal of this project was to bridge the gap between raw data and business decision-making by solving key analytical problems:

### 🔹 Problem 1: Lack of visibility into overall business performance
- Solution: Calculated total orders and total revenue to understand scale

### 🔹 Problem 2: Identifying high-value products
- Solution: Analyzed highest-priced pizzas and top revenue-generating pizza types

### 🔹 Problem 3: Understanding customer preferences
- Solution: Identified most common pizza sizes and most ordered pizza types

### 🔹 Problem 4: Category-level performance is unclear
- Solution: Evaluated category-wise sales distribution and contribution to revenue

### 🔹 Problem 5: No insight into demand patterns over time
- Solution: Analyzed order distribution by hour and calculated cumulative revenue trends

### 🔹 Problem 6: Need for actionable insights for decision-making
- Solution: Identified top 3 pizzas overall and within each category using advanced SQL (window functions)

---

## 🚀 Approach

- Used SQL joins to combine multiple tables (orders, pizzas, pizza types)
- Applied aggregate functions (SUM, COUNT, AVG) for business metrics
- Used window functions (RANK, cumulative SUM) for advanced analysis
- Structured queries to answer real-world business questions

## 💡 Outcome
